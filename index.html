02

Diagrama Causa-Efecto

Análisis de Ishikawa — identificación de causas raíz antes y después de mejoras.

Diagrama Ishikawa

Produccion lenta y piezas con mal acabado

Resultado: baja produccion + alto costo + calidad irregular

Mano de Obra

  • - Trabajo manual en casi todas las operaciones
  • - Mucho tiempo dedicado al pulido
  • - Fatiga por tareas repetitivas
  • - Resultados dependen del operador
  • - Bajo interes en el trabajo

Metodo

  • - Operaciones sin orden definido
  • - Reprocesos por acabado superficial
  • - Ajustes improvisados durante fabricacion
  • - Flujo interrumpido entre estaciones
  • - Tiempos de produccion muy largos

Maquinaria

  • - Corte mecanico tradicional
  • - Precision limitada en formas complejas
  • - Bajo rendimiento del equipo
  • - Necesidad constante de intervencion humana
  • - Produccion inestable

Materiales

  • - Acero inoxidable dificil de maquinar
  • - Rebabas despues del corte
  • - Piezas requieren retrabajo
  • - Variacion en el acabado
  • - Desperdicio de material

Medio Ambiente

  • - Lay-out desordenado
  • - Traslados innecesarios de piezas
  • - Iluminacion deficiente
  • - Falta de seguridad
  • - Espacios poco organizados

Medicion

  • - Inspeccion solo al final
  • - Defectos detectados tarde
  • - Sin control de tiempos
  • - Criterio visual subjetivo
  • - No se registran datos del proceso

Ishikawa 2 — Despues del Proyecto (Situacion mejorada)

Produccion continua de dijes deportivos con calidad uniforme

Resultado: mayor volumen + menor costo + calidad constante

Mano de Obra

  • + Operador supervisa el sistema
  • + Menor esfuerzo fisico
  • + Trabajo mas ergonomico
  • + Menos errores humanos
  • + Mayor motivacion

Metodo

  • + Proceso definido digitalmente
  • + Secuencia fija de operaciones
  • + Eliminacion de reprocesos
  • + Flujo continuo
  • + Produccion planificada

Maquinaria

  • + Equipos automatizados
  • + Precision en geometrias complejas
  • + Parametros programados
  • + Produccion estable
  • + Menos intervencion manual

Materiales

  • + Corte limpio desde el inicio
  • + Menor desperdicio
  • + Acabado uniforme
  • + Piezas listas para ensamble
  • + Aprovechamiento del material

Medio Ambiente

  • + Lay-out optimizado
  • + Menos transporte interno
  • + Iluminacion adecuada
  • + Areas definidas
  • + Seguridad industrial

Medicion

  • + Control durante el proceso
  • + Registro de produccion
  • + Parametros definidos
  • + Deteccion temprana de fallas
  • + Seguimiento de calidad

Fichas Técnicas de Seguridad

Controles de riesgo, EPP, señalización y procedimientos seguros por estación de trabajo.

Área de Materia Prima

Medio

Riesgos

  • • Cortes por bordes afilados
  • • Golpes por caída/deslizamiento de láminas
  • • Sobreesfuerzo por manipulación manual
  • • Atrapamiento leve entre láminas/racks

Controles de Ingeniería

  • • Racks anclados y en buen estado
  • • Separadores/topes antideslizamiento
  • • Piso antiderrapante con delimitación del área

Controles Administrativos

  • • Capacitación en manejo manual y técnicas de levantamiento
  • • Límite de carga visible en cada rack
  • • Implementación de metodología 5S y pasillos libres de obstáculos
  • • Prohibido manipular material sin guantes de protección

EPP

  • • Guantes anticorte
  • • Lentes de seguridad
  • • Botas de seguridad con casquillo

Señalización

  • • Material cortante (amarillo)
  • • Uso obligatorio guantes y lentes
  • • Delimitación del área de almacenamiento
  • • Ruta peatonal señalizada

Procedimiento Seguro de Operación

  1. Verificar que el área de trabajo se encuentre despejada
  2. Colocar todo el EPP antes de manipular material
  3. Manipular las láminas utilizando técnicas seguras de levantamiento
  4. No sobrecargar los racks de almacenamiento
  5. Reportar material dañado
